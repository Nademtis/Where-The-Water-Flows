# Where The Water Flows

Is an 2D isometric puzzle platformer made for the GameOff2025 Game Jam.\
[Play here on itch.io](https://nademtis.itch.io/gameoff2025-where-the-water-flows) in the browser
## Known Issues

- Mac users might experience visuals glitches... please reach out if you encounter these glitches, and know how to fix them
- Can't load the game? - try running in Chrome


### About
This game was made in 30 days for the GameOff2025 game jam, with the given theme Waves.

### TechStack
- Godot 4.5.1
  - PhantomCamera
  - Aseprite Wizard
- Aseprite
- Ableton Live 12
- Splice
- Tiled
